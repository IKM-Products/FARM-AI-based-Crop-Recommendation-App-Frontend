# 🌱 FARM: AI-Based Crop Recommendation App

**FARM** is an intelligent agricultural mobile application that helps farmers and agricultural enthusiasts make informed decisions about which crops to grow based on environmental and soil conditions. By leveraging Machine Learning and real-time weather data, the app provides accurate crop recommendations to improve productivity and sustainability.

## 🚀 Features

* 🌾 **AI-Powered Crop Recommendation**
  Suggests the most suitable crops based on input parameters like temperature, humidity, rainfall, and soil nutrients.

* 🌍 **Real-Time Weather Integration**
  Fetches live environmental data (temperature, humidity, rainfall) using weather APIs for accurate predictions.

* 🧪 **Soil Parameter Input**
  Users can manually input soil data such as:

  * Nitrogen (N)
  * Phosphorus (P)
  * Potassium (K)
  * pH value

* 📊 **Machine Learning Model**
  Trained on agricultural datasets to provide optimized and data-driven crop suggestions.

* 📱 **User-Friendly Interface**
  Simple and intuitive design for easy use by farmers and general users.

## 🛠️ Technologies Used

* **Frontend:** Android (Java/XML) or Flutter (Dart)
* **Backend/Logic:** Python (Machine Learning Model)
* **ML Algorithm:** (e.g., Decision Tree / Random Forest / KNN – customizable)
* **API:** Open-Meteo API (for weather data)
* **Dataset Source:** Kaggle agricultural datasets

## 📱 How It Works

1. User enters soil nutrient values (N, P, K, pH).
2. App fetches real-time weather data (temperature, humidity, rainfall).
3. The ML model processes the input data.
4. The app recommends the most suitable crop(s) to grow.
5. User can make better farming decisions based on recommendations.

## 🎯 Objective

The goal of **FARM** is to empower farmers with smart technology by providing accurate crop recommendations, reducing guesswork, and increasing agricultural efficiency and yield.
